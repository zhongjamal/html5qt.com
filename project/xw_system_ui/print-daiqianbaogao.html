小窝金服-工行家装贷商家合作协议

协议编号: XWJF-SJ-2015-

本协议由以下双方共同缔结：

  • 1、甲方：深圳市小窝金融信息服务有限公司 （下称 “甲方”或“小窝金服”）
  • 2、乙方：（下称“商户”或“乙方”）

就工行家装贷业务合作的相关事宜，本着遵循平等、自愿、公平诚信的原则，甲乙双方友好协商，秉着共同发展、诚信合作 的宗旨达成如下协议：

一、权利和义务

1.1双方共同开发家装贷客户市场并进行客户转化。

1.2甲方为双方合作客户提供贷款申请、工程验真、资金托管，监督和协助验收装修项目，同时依据《工行家装贷服务合作协议》约定的工程各支付节点按照约定比例支付工程款给乙方。

1.3乙方负责双方合作客户的装修工程施工及工程保维修。

二、服务佣金和合作保证金

2.1合作商家推荐签约的家装贷客户，不收取合作商家合作保证金和佣金。

2.2甲方推荐签约客户，为保障消费者与甲方的权益，乙方应向甲方支付合作保证金和佣金；服务佣金的支付方式是甲方按每笔工行家装贷服务订单的托管总额的5%的标准，按《工行家装贷服务合作协议》约定的支付节点按照约定比例从乙方应收工程款项中主动划扣；保证金的支付方式是甲方主动从首个工行家装贷服务项目的款项中扣划，合作保证金为人民币 元整（RMB￥：元整）。

2.3保证金的返还

协议终止后，同时符合以下条件的，甲方应向乙方无息返还保证金：

（1）协议期内乙方不存在根据本协议、小窝金服其它规定应当使用保证金向小窝金服和/或消费者支付违约金、赔偿金或补偿金的情形；且，

（2）乙方与甲方有关服务费用均已结清；且， （3）至协议终止后三（3）个月内，乙方未受到任何第三方投诉或发生交易纠纷。

（3）至协议终止后三（3）个月内，乙方未受到任何第三方投诉或发生交易纠纷。

甲方将在上述第（3）项期限届满后的五（5）个工作日内向乙方返还剩余的保证金。

三、协议的解除与终止

4.1 自然终止：本协议自约定的期限届满终止。本协议期限为壹年，自协议签定之日起算。

4.2 约定解除：本协议任何一方必须提前十五天书面通知的方式终止本协议，书面通知到达对方生效。

四、违约责任

5.1 乙方违反约定单方解除协议的，乙方应按约定承担违约责任，并赔偿甲方的损失。

5.2 乙方承诺，不会采取任何手段或措施隐瞒经甲方提供的服务达成的交易，以规避向甲方支付相关服务费用或规避甲方对工行家装贷业务的监管。否则将视为严重违约，甲方可将其保证金作为违约金予以划扣，并有权立即终止本协议。

五、其他

6.1本协议的任何变更均应以书面形成，经双方签署后方生效；本协议一式叁份，甲方执二份，乙方执一份，均具有同等法律效力。
